# Smart Lender

## Project Overview
Smart Lender is a machine learning application that predicts whether a loan application will be approved based on applicant details such as income, education, credit history, and other factors.

## Dataset
- Dataset: Loan Prediction Dataset
- Source: Google Sheets / Provided project link

## Features
- Data analysis and visualization
- Data preprocessing
- Decision Tree model
- Random Forest model
- Loan approval prediction
